## Runs Locally


### Clone the repository and then install and run with npm

```bash
git clone "https://github.com/Shad0wCrux/character-animation-combiner"
cd character-animation-combiner/

npm install

npm run dev
```

### The app will be running in development mode.<br />
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
